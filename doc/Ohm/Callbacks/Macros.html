- (Array?) after(method, callback)

Use to add an after callback on `method`. Only symbols are allowed, no string eval, no block option also.

Examples:

class Post < Ohm::Model
  include Ohm::Callbacks

  after  :create, :post_to_twitter!
  after  :save,   :sync_ids

protected
  def post_to_twitter!
    # do twitter posting here
  end

  def sync_ids
    # do something with the ids
  end
end

Parameters:

  • (Symbol) method —

    the method type, `:validate`, `:create`, or `:save`

  • (Symbol) callback —

    the name of the method to execute

Returns:

  • (Array) —

    the callback in an array if added e.g. [:timestamp]

  • (nil) —

    if the callback already exists

def after(method, callback)
  unless callbacks[:after][method].include? callback
    callbacks[:after][method] << callback
  end
end
